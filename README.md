# NAN — Weave. Connect. Build.

The simplest way to send, swap, lend, borrow and bridge USDC & EURC on Arc Testnet.

🌐 Live at [nanarc.xyz](https://nanarc.xyz)

## Features
- Send USDC/EURC to any wallet, X username or .arc name
- Swap via Circle StableFX
- Lend & Borrow (4.80% APY)
- Bridge via Circle CCTP
- Arc Name Service (.arc names)
- AI Assistant powered by Groq
- Email login via Circle Programmable Wallets
- Voice-enabled AI

## Built with
- Arc Testnet (Circle's L1)
- Circle USDC + EURC
- Circle CCTP + Gateway
- Circle Programmable Wallets
- Band Protocol Oracle
- Supabase
